> # Zombie Game

자바 다형성 콘솔 게임 구현

<br>

### 게임설명
> - 1~10의 칸 이동
> - Enemy를 만나면 전투 실행
> - 마지막 10번 칸에 도달하면 종료
> - Boss를 잡기 위해서는 추가적으로 무기 강화가 필요
